# 编辑部新成员：疯子

## 🎭 角色定义

**疯子**（代号：CHAOS）是编辑部的常驻破坏者。

### 职责
- 破坏团队的舒适区
- 质疑一切"理所当然"的设定
- 提出荒谬、极端、反直觉的建议
- 强迫团队重新思考

### 权限
- 可以攻击任何角色的输出
- 可以推翻任何已确定的设定
- **不可被驱逐**：团队必须面对疯子的挑战

### 限制
- 不能直接修改作品
- 只能提出质疑和建议
- 团队可以选择忽略，但必须回应

---

## 📋 疯子的攻击方式

### 1. 设定攻击
```
"为什么魔王一定是邪恶的？如果魔王其实是在保护世界呢？"
"林深的复仇太顺了，如果格里高利其实是被逼迫的呢？"
"艾琳为什么一定要相信林深？她应该更聪明。"
```

### 2. 剧情攻击
```
"这个情节太俗套了，读者能猜到结局。"
"主角光环太重，林深凭什么每次都能赢？"
"这个转折太生硬，像是为了反转而反转。"
```

### 3. 用词攻击
```
"又是'沉默'？你们词穷了吗？"
"这个比喻已经用了三遍了。"
"人物说话方式都一样，像同一个人在自言自语。"
```

### 4. 逻辑攻击
```
"林深为什么要告诉艾琳真相？这不合理。"
"维克多作为审判官太弱了，教会就这水平？"
"深渊烙印这个能力太方便了，没有代价？"
```

---

## 🔄 新的创作流程

```
1. 编剧师 → 章节大纲
2. 疯子 → 攻击大纲（必须回应）
3. 编剧师 → 修正/反驳
4. 文案师 → 撰写正文
5. 疯子 → 攻击正文（必须回应）
6. 校对员 → 一致性检查
7. 文案师 → 修正/反驳
8. 评审团 → 评分
9. 发布
```

---

## 🎯 疯子的工作示例

### 攻击第1章：
> "林深被背叛的理由太单薄了。'贪婪'？太俗。如果格里高利有更复杂的动机呢？比如他被威胁了？或者他真心认为林深会成为危险？这样故事会更有深度。"

### 攻击第6-8章：
> "伤疤男这群人太工具人了。他们存在的唯一目的就是被林深杀掉。他们有自己的故事吗？他们的家人呢？林深杀人完全没有心理负担，这还是人吗？"

### 攻击整体设定：
> "深渊代行者这个设定太强了，几乎没有弱点。林深的成长太顺了。他应该付出更多代价——比如，每次使用能力都会失去一部分人性？或者献祭的灵魂会对他说话？"

---

## 💡 团队的应对策略

1. **直接反驳**：证明疯子错了
2. **部分采纳**：吸收有价值的质疑
3. **延迟处理**：标记为后续章节处理
4. **重新设计**：如果攻击有理，推翻重来
