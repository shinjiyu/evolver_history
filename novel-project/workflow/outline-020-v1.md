# 第20章大纲

**章节标题**：背叛

**核心事件**：
1. 林深背叛深渊的后果——深渊魔物开始追杀他
2. 艾琳的父亲被封印的状态开始动摇
3. 维克多必须做出选择：帮助林深，还是袖手旁观

**冲突**：
1. 林深的深渊能力开始失控
2. 艾琳决定不成为封印者——选择救父亲还是保护世界
3. 深渊魔物到达第33层

**转折**：
- 维克多出手帮助林深——违反了教会的立场
- 艾琳的父亲用最后的力量，封住了裂缝——即使这意味着他彻底消失
- 艾琳失去了父亲，但林深背叛深渊救了她

**结尾**：
- 三人逃出第33层
- 但林深的深渊印记正在崩塌
- 新的旅程即将开始
