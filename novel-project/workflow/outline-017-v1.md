# 第17章大纲

**章节标题**：深入

**核心事件**：
1. 林深、艾琳、维克多三人进入地下城
2. 他们遇到第10层的危险（石像鬼群）
3. 维克多第一次看到林深使用深渊能力

**冲突**：
1. 三人之间的不信任
2. 艾琳质疑维克多的立场
3. 维克多目睹深渊能力后陷入矛盾

**转折**：
- 在石像鬼攻击中，林深救了维克多
- 维克多问："你为什么不让我死？"
- 林深回答："因为观察者说，我的选择会决定艾琳的命运。而你，对艾琳很重要。"

**结尾**：
- 三人通过第10层，继续深入
- 但更深的危险正在等待他们
