# 第16章大纲

**章节标题**：三方

**核心事件**：
1. 教会召开紧急会议，讨论"观察者"的问题
2. 维克多被要求同时调查林深和观察者
3. 艾琳独自行动，准备进入地下城

**冲突**：
1. 教会内部对如何处理"虚空"存在分歧
2. 维克多需要在"服从命令"和"自己判断"之间选择
3. 林深发现艾琳独自进入地下城，需要决定是否跟随

**转折**：
- 维克多私下找到林深
- "我知道你在隐藏什么。"他说，"但我需要你的帮助。"
- 教会怀疑观察者是"虚空代理人"，但他们不知道虚空是什么

**结尾**：
- 林深和维克多达成临时协议
- 他们将一起追踪艾琳
- 但他们都知道：这只是暂时的合作
