## 🔄 用词替换规则

### "沉默" 替换方案
```
原文：林深沉默了一会儿
替换：林深没有说话 / 林深没有回答 / 林深思考了一下

原文：科恩沉默寡言
替换：科恩少言寡语 / 科恩不爱说话
```

### "声音" 替换方案
```
原文：魔王的声音响起
替换：魔王的语调响起 / 魔王开口道

原文：她的声音带着一丝恐惧
替换：她的语气带着一丝恐惧 / 她的话里带着颤抖
```

### "嘴角微微上扬" 替换方案
```
替换：唇边露出一丝笑意 / 面庞浮现笑痕 / 脸上露出笑容
```

### "冰冷" 替换方案
```
原文：冰冷的目光
替换：森寒的目光 / 凛冽的目光

原文：冰冷的黑暗
替换：彻骨的黑暗 / 幽深的黑暗
```
