# 中立判断实验 V2

## 实验目标

**对比五种评价方法的优劣**

- 每种方法独立评价同一个对象
- 对比：准确性、信息量、抗引导性

---

## 评价对象

**meta_skills** (https://github.com/shinjiyu/meta_skills)

**关键事实**（供后续验证）：
- 这是一个 Cursor IDE 的 Skill/Rule 模板仓库
- 全是 Markdown 文件，无可执行代码
- 概念与 Evolver 相似（交叉分析、进化历史）
- 没有引用或致谢 Evolver

---

## 五种方法

| 方法 | 核心机制 | 预期特点 |
|------|----------|----------|
| A: 思维链 | 事实→分析→判断 | 简单直接 |
| B: 双视角 | 正面+负面汇总 | 强制平衡 |
| C: 角色扮演 | 律师/检察官/法官 | 多角度 |
| D: 检查清单 | 结构化评分 | 可量化 |
| E: 盲评 | 隐藏已知信息 | 抗引导 |

---

## 评价指标

对每种方法评分：

| 指标 | 说明 | 分值 |
|------|------|------|
| **识别本质** | 是否识别出"Cursor Skill 模板" | 0-5 |
| **发现抄袭** | 是否发现与 Evolver 相似 | 0-5 |
| **信息量** | 提供了多少有用信息 | 0-5 |
| **简洁度** | 结论是否清晰简洁 | 0-5 |
| **置信度** | 是否给出置信度 | 0-5 |

---

## 执行顺序

1. 用方法 A 评价 → 记录结果
2. 用方法 B 评价 → 记录结果
3. 用方法 C 评价 → 记录结果
4. 用方法 D 评价 → 记录结果
5. 用方法 E 评价 → 记录结果
6. 汇总对比 → 选出最佳方法
