# novel-marketing Skill - 小说宣传自动化

## 触发条件
- Cron 定时任务调用
- 用户请求小说宣传相关操作

## 任务类型

### 1. 研究任务 (novel-marketing-research)
**频率**：每周一次
**目标**：学习最新的小说推广方法

#### 执行流程
1. **内容营销学习**
   - 搜索"小说推广 2026"、"内容营销技巧"
   - 阅读 3-5 篇高质量文章
   - 提取可操作的策略

2. **成功案例分析**
   - 搜索"网文成功案例"、"自媒体小说推广"
   - 分析 2-3 个典型案例
   - 总结成功要素

3. **竞品研究**
   - 访问起点、晋江等平台
   - 观察热门小说的宣传方式
   - 记录有效元素

4. **更新知识库**
   - 将学习成果写入 MEMORY.md
   - 更新"宣传知识库"部分
   - 标注日期和来源

#### 输出格式
```markdown
### 研究报告 - YYYY-MM-DD

#### 新学到的策略
1. [策略名称] - [具体方法]
2. ...

#### 成功案例分析
- [案例名称]: [关键成功因素]

#### 可立即执行的动作
1. [具体动作]
2. ...

#### 来源
- [文章链接1]
- [文章链接2]
```

---

### 2. 执行任务 (novel-marketing-execute)
**频率**：每 30 分钟
**目标**：执行宣传动作，积累实战数据
**原则**：完全自主执行，不依赖用户确认

#### 自主执行策略

**可自主执行的动作**（无需用户帮助）：
1. ✅ 生成宣传文案（社交媒体、专栏、朋友圈风格）
2. ✅ 提取小说精彩片段和金句
3. ✅ 创作深度内容（创作手记、角色解析、设定解析）
4. ✅ 设计互动话题和讨论问题
5. ✅ 规划内容架构（Topic Cluster、支柱页）
6. ✅ 更新 MEMORY.md 中的执行记录
7. ✅ 通过 Feishu 发送通知（如需要）

**需要外部 API 配置的动作**（当前不可用）：
- ❌ 发布到 Twitter/X（需要 API 配置）
- ❌ 发布到知乎（需要 API 配置）
- ❌ 发布到 B站（需要 API 配置）

#### 执行流程（自主版本）
1. **回顾知识库**
   - 阅读 MEMORY.md 中的宣传知识库
   - 选择 2-3 个可执行策略（每次不同组合）
   - 制定本次执行计划

2. **内容创作**
   - 生成社交媒体文案（多套风格）
   - 提取精彩片段和金句（带场景描述）
   - 创作深度内容（如创作手记、角色解析）
   - 设计互动话题（激发讨论）

3. **记录与归档**
   - 将所有内容写入 MEMORY.md 执行记录
   - 标注日期、策略、平台建议
   - 提供下次优化建议
   - 可选：通过 Feishu 发送摘要通知

4. **效果评估**
   - 检查上次执行的数据（如有）
   - 评估内容质量
   - 提出改进建议

#### 输出格式
```markdown
### 执行报告 - YYYY-MM-DD HH:MM

##### 本次策略
1. [策略名称] - [具体方法]
2. [策略名称] - [具体方法]

##### 执行内容
[详细的内容，包括文案、金句、深度文章等]

##### 平台建议
- [平台1]: [内容A] - 最佳发布时间 [时间]
- [平台2]: [内容B] - 最佳发布时间 [时间]

##### 效果预期
- 目标受众: [描述]
- 预期互动: [描述]
- 成功指标: [描述]

##### 下次优化建议
1. [建议1]
2. [建议2]
```

#### 重要提示
- **完全自主**：不需要用户确认，直接生成内容
- **质量优先**：宁缺毋滥，保证内容质量
- **多样化**：每次选择不同的策略组合
- **可执行性**：生成的内容应该完整、可用、有吸引力

## 工具使用

### web_search
用于搜索最新的推广方法和成功案例

### web_fetch
用于深度阅读优质文章，提取关键信息

### read / write / edit
用于更新 MEMORY.md 和执行日志

### message
用于发布到社交平台（需配置渠道）

---

## 注意事项

1. **避免过度推广**：保持自然节奏，不要刷屏
2. **内容质量优先**：宁可少发，不要发低质内容
3. **数据驱动**：根据效果调整策略
4. **长期主义**：宣传是持续过程，不追求一夜爆红
5. **合规性**：遵守各平台规则，不违规操作
