# auto-platform-registration Skill - 自动平台注册学习

## 任务目标
持续学习、收集和实践各种平台的自动注册方法

## 执行流程

### 研究阶段
1. 搜索关键词：自动注册、平台账号、API 注册、自动化工具
2. 关注平台类型：AI 平台、社交媒体、开发平台、服务类平台
3. 学习内容：注册流程、验证方式、CAPTCHA 处理、邮箱验证、手机验证

### 实践阶段
1. 分析已收集的注册方法
2. 识别可自动化环节
3. 测试自动化脚本
4. 记录成功率

### 记录阶段
更新 /root/.openclaw/workspace/MEMORY.md 中的"自动注册知识库"
