### C. 生成修复脚本（1 个）

#### Gateway 内存监控脚本

**文件**: `/root/.openclaw/workspace/evolver/fixes/gateway-memory-monitor.sh`

**核心功能**:

1. **实时监控**
   - 获取 Gateway 进程信息
   - 监控内存使用（MB 和百分比）
   - 监控 CPU 使用率
   - 显示运行时间

2. **历史记录**
   - CSV 格式存储历史数据
   - 自动记录时间戳
   - 便于趋势分析

3. **趋势分析**
   - 计算内存增长率
   - 识别内存泄漏风险
   - 显示平均内存使用

4. **优化建议**
   - 根据内存状态提供建议
   - 推荐重启时机
   - 提供自动化配置命令

**四种模式**:

1. **check** - 单次检查（默认）
   ```bash
   ./gateway-memory-monitor.sh check
   ```

2. **watch** - 持续监控（每 5 分钟）
   ```bash
   ./gateway-memory-monitor.sh watch
   ```

3. **quiet** - 静默模式（用于 cron）
   ```bash
   ./gateway-memory-monitor.sh quiet
   ```

4. **history** - 查看历史记录
   ```bash
   ./gateway-memory-monitor.sh history
   ```

**告警阈值**:
- 危险: > 1500MB
- 警告: > 1200MB
- 正常: < 1200MB
