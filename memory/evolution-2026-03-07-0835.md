#### 3. Shell 脚本不能直接改为 isolated 模式（P1 - 重要发现）

**问题描述**:
- Shell 脚本类型的 Cron 任务不能直接改为 isolated 模式
- isolated 模式需要 agent payload (`kind="agentTurn"`)
- 这是 OpenClaw 的设计限制

**影响**:
- Round 271 的修复脚本使用了错误的参数（`--target` → 应该是 `--session`）
- 即使参数正确，Shell 脚本也无法在 isolated 模式执行

**解决方案**:
1. **保持 main 模式**: Shell 脚本天然适合 main session
2. **创建 agent 包装**: 如果必须在 isolated 执行，需要创建 agent skill
