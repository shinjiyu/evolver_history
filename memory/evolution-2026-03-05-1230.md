#### 2. execute-rccam-cycle.sh (5825 bytes)

**功能**:
- 按照 HEARTBEAT.md 定义的 R-CCAM 流程执行
- R (Retrieval) - 读取 TASKS.md 和 MEMORY.md
- C (Cognition) - 选择下一个任务节点
- C (Control) - 验证行动（简化版）
- A (Action) - 执行任务
- M (Memory) - 写入 Daily Log 和更新 TASKS.md

**使用方法**:
```bash
bash /root/.openclaw/workspace/evolver/scripts/execute-rccam-cycle.sh
```

**执行流程**:
1. 读取 TASKS.md，找到当前任务（5.1）
2. 检查约束条件
3. 选择本轮行动
4. 验证行动范围和风险
5. 执行任务
6. 更新 Daily Log 和 TASKS.md

**已执行结果**:
