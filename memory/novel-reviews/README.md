# 小说自动评审系统

## 📚 系统概述

这是一个全自动的小说评审系统，每30分钟自动执行一次评审任务。

## 🎯 四种评审模式（轮换执行）

### 模式1：连续章节一致性评审
- **目标**：检查2个连续章节的前后一致性
- **团队角色**：
  - 逻辑审核员 - 检查情节逻辑和因果关系
  - 设定管理员 - 检查世界观和设定一致性
  - 人物分析师 - 检查人物行为和动机
- **协作流程**：并行分析 → 汇总问题 → 提出修改方案

### 模式2：长距离章节一致性评审
- **目标**：检查相距较远章节的设定演变和人物成长
- **团队角色**：
  - 设定考古学家 - 追踪设定演变
  - 人物成长分析师 - 分析人物弧光
  - 伏笔管理员 - 检查伏笔和呼应
- **协作流程**：并行分析 → 交叉验证 → 综合报告

### 模式3：灵感评审
- **目标**：提供创意改进建议
- **团队角色**：
  - 创意顾问 - 提供情节创新建议
  - 对话优化师 - 优化对话表现
  - 场景设计师 - 强化场景效果
- **协作流程**：头脑风暴 → 筛选建议 → 优先级排序

### 模式4：常规文学评审
- **目标**：检查文笔、节奏、对话、人物塑造
- **团队角色**：
  - 文字编辑 - 检查用词和语法
  - 节奏分析师 - 分析叙事节奏
  - 人物塑造顾问 - 评估人物表现
- **协作流程**：并行评审 → 汇总问题 → 修改建议

## 🚀 使用方法

### 1. 手动执行（测试）
```bash

# 生成评审任务（dry run）
```

### 2. 查看评审任务
```bash

```

```
2. 读取评审状态（state.json）
   ↓
3. 选择下一个评审模式（4种轮换）
   ↓
4. 选择评审章节
   ↓
5. 生成评审提示词
   ↓

   ↓
7. 保存评审任务
   ↓
8. 更新评审状态
   ↓
9. 等待下一次执行（30分钟后）
```

## 🔧 自定义配置

### 修改评审频率
```bash
# 删除现有任务
openclaw cron delete 146083a3-dae7-409d-86cc-6a953d1cbdd6

# 创建新任务（例如：每1小时）
openclaw cron create \
  --name "novel-auto-review" \
  --every "1h" \
  --message "执行小说自动评审。请运行: cd /root/.openclaw/workspace && node cron/novel-auto-review-full.js" \
  --session "isolated" \
  --description "小说自动评审系统 - 四种模式轮换执行"
```

### 修改评审范围
编辑 `cron/novel-auto-review-full.js` 中的 `CONFIG.maxChapters`（默认60）

### 添加新的评审模式
编辑 `cron/novel-auto-review-full.js` 中的 `REVIEW_MODES` 数组
