## 六、核心修改示例

### 示例1：塞缪尔形象优化

**原文**：
```
"当然。"塞缪尔微笑，"艾琳是我们最好的诱饵。"
```

**优化后**：
```
"当然。"塞缪尔的声音没有丝毫温度，"艾琳是我们最好的诱饵。你以为我会为此愧疚？三十年来，我看着教廷的腐败，看着他们用'神圣'的名义犯下罪行……深渊至少诚实。"
```

### 示例2：艾琳觉醒触发条件

**原文**：
```
就在这时，牢房里的艾琳突然睁开了眼睛。
```

**优化后**：
```
就在这时，牢房里的艾琳突然睁开了眼睛。

林深危险。

这个念头在她混沌的意识中炸开，像一道闪电撕裂黑暗。她的血脉——那个她一直逃避的血脉——在此刻做出了选择。

她的眼睛变成了纯紫色。
```

### 示例3：莉迪亚个性台词

**原文**：
```
"够了。"莉迪亚说，"别和他废话。深渊要他活着，但没有说他必须完整。"
```

**优化后**：
```
"够了。"莉迪亚的声音像刀子一样锋利，"我最讨厌话多的男人。深渊要他活着，但没有说他必须完整。"
```
