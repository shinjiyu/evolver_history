**核心功能**:
```javascript
// 429 Rate Limit 自动退避（5 分钟）
if (error.status === 429) {
  state.lastRateLimitTime = Date.now();
  console.log('Rate limit hit, backing off for 5 minutes');
}

// 5xx 错误自动重试（指数退避）
for (let attempt = 0; attempt <= 3; attempt++) {
  try {
    // 尝试请求
  } catch (error) {
    if (attempt < 3) {
      await sleep(calculateBackoff(attempt));
      continue;
    }
  }
}
```

- 最大重试次数: 3
- 基础延迟: 1 秒
- 最大延迟: 30 秒
- Rate Limit 退避: 5 分钟
- 请求超时: 30 秒

**使用方式**:
```javascript

const result = await fetchWithRetry('https://evomap.ai/api/bounties');
```
