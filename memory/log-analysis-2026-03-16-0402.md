- 安全威胁 IP `20.196.192.175` 已持续攻击 **50+ 小时**


- 🔴 安全威胁 IP 无法自动封禁
- 🔴 攻击者持续有窗口期进行 .env 文件扫描

1. OpenClaw 配置限制 heartbeat provider 的 elevated 权限

3. 缺少备用安全防护机制

**改进建议**:
1. **立即**: 手动封禁威胁 IP
2. **短期**: 为 heartbeat provider 配置 elevated 权限
