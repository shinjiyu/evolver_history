**严重程度**: P1 (高)  
**出现次数**: 多次  
**影响范围**: EvoMap 节点状态监控

**问题描述**:


1. 增加心跳超时时间（当前可能是 30s，建议增加到 60s）
2. 添加重试机制（失败后等待 5s 重试一次）

### P2 - 文件编辑失败：找不到确切文本

**严重程度**: P2 (中)  
**出现次数**: 4 次  
**影响范围**: 自动化编辑任务

**问题描述**:
在尝试编辑文件时，系统无法找到指定的文本，导致编辑操作失败。

**错误示例**:
```json
{"error":"Could not find the exact text in /root/.openclaw/workspace/memory/2026-03-01.md. The old text must match exactly including all whitespace and newlines."}
{"error":"Could not find the exact text in /root/.openclaw/workspace/HEARTBEAT.md. The old text must match exactly including all whitespace and newlines."}
{"error":"Could not find the exact text in /root/.openclaw/workspace/evolver/pattern-registry.md. The old text must match exactly including all whitespace and newlines."}
```

**根本原因**:
1. 文件内容在编辑请求生成和实际执行之间发生了变化
2. 多个任务同时编辑同一文件导致竞争条件
3. 文本匹配过于严格（要求精确匹配包括空格和换行）

**改进建议**:
1. 在编辑前重新读取文件内容，确保文本仍然存在
2. 使用更宽松的匹配策略（如正则表达式或模糊匹配）
3. 实现文件锁机制，避免并发编辑冲突
4. 添加编辑失败后的自动重试逻辑

### P2 - EvoMap 资产发布失败

**严重程度**: P2 (中)  
**出现次数**: 多次  
**影响范围**: EvoMap 资产发布流程

**问题描述**:
EvoMap Capsule 发布任务频繁失败，返回多种 HTTP 错误。

**错误示例**:
```
❌ 发布失败: HTTP 503: {"error":"network_frozen: unable to verify network status. Rejecting publish as a safety precaution."}
❌ 发布失败: HTTP 400: {"error":"gene_strategy_step_too_short: each step must be at least 15 characters describing an actionable operation"}
```

**根本原因**:
1. HTTP 503: EvoMap 网络状态检查失败，服务器拒绝发布
2. HTTP 400: Gene 策略步骤描述太短（少于 15 字符）
3. HTTP 409: 任务队列已满

**改进建议**:
1. 发布前检查网络状态，避免在 frozen 状态下尝试发布
2. 验证 Gene 策略步骤长度，确保每个步骤至少 15 字符
3. 实现指数退避重试机制
4. 添加发布前预检查脚本
