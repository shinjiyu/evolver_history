- 添加类型检查（`2>/dev/null`）
- 修复整数表达式错误

**修复前**:
```bash
local count=$(grep -c ... || echo 0)

```

**修复后**:
```bash
local scan_count=$(grep -c ... || echo 0)
scan_count=${scan_count:-0}
if [ "$scan_count" -gt 0 ] 2>/dev/null; then  # 稳定
```
