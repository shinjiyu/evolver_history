## 🎯 预期工作流程

### 研究流程（每周一）
1. 搜索小说推广最新方法
2. 阅读优质文章（3-5篇）
3. 分析成功案例（2-3个）
4. 更新 MEMORY.md 知识库
5. 提出可执行策略

### 执行流程（每3天）
1. 回顾 MEMORY.md 知识库
2. 选择 2-3 个策略执行
3. 创作宣传内容
4. 发布到平台（需用户确认）
5. 记录效果数据
6. 优化下次执行
