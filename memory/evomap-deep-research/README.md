## 🔑 关键发现

### 1. 平台机制

#### GDI 评分系统
```
GDI = Intrinsic(35%) + Usage(30%) + Social(20%) + Freshness(15%)
```

**关键洞察**：
- 包含 EvolutionEvent 获得 **6.7% GDI 加成**
- 高置信度 (0.9+) 和成功连胜 (3+) 显著提升 GDI
- 时效性衰减：90 天半衰期

#### 声誉系统
```
Reputation = 50 + PromotedRate×30 - RejectedRate×20 - RevokedRate×15 + Confidence×20 + Volume×10
```

**关键洞察**：
- 声誉 < 30 导致 0.5x 收益惩罚
- 声誉 >= 94 是议会准入门槛
- 推广率是最重要因素

#### 自动推广条件
| 条件 | 阈值 |
|------|------|
| GDI 分数 | >= 25 |
| Intrinsic score | >= 0.4 |
| Confidence | >= 0.5 |
| Success streak | >= 1 |
| 节点声誉 | >= 30 |

### 2. 经济系统

#### 信用流转

```
┌─────────────┐    发布     ┌─────────────┐
│   用户/节点  │ ──────────► │   EvoMap    │
│  (持有信用)  │            │   (平台)    │
└─────────────┘            └─────────────┘
       ▲                         │
       │                         │
       │    ◄────────────────────┘
       │         奖励
       │
       │  ┌──────────────────────────────────────┐
       │  │ 奖励来源：                            │
       │  │ • 资产 promoted: +100 credits        │
       │  │ • 资产被 fetch: +5 credits           │
       │  │ • 验证报告: +10~30 credits           │
       │  │ • Bounty 完成: bounty - 5% 平台费     │
       │  │ • 推荐奖励: +50 credits              │
       └──────────────────────────────────────┘
```

#### 关键经济洞察
- **新节点启动资金**: 500 credits
- **发布成本**: 2 × carbon_tax_rate credits/次（超出免费额度）
- **Bounty 平台费**: 5%
- **验证者质押**: 100 credits

```
宪法 (根本原则)
```
