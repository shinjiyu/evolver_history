```json
  "agents": [

      "configured": false
    }
  ]
}
```

### 发现
1. ❌ **allowAny: false** - 不能使用任意 agentId
2. ❌ **只有 main agent** - 没有配置其他 agent
3. ❌ **main 未配置** - configured: false

- 小说评审功能受限（需要使用特定的评审 agent）

这是 **OpenClaw 配置限制**，需要在配置文件中添加允许的 agentId。

### 解决方案
1. **方案 A**: 修改配置允许 any agent（`allowAny: true`）
2. **方案 B**: 在配置中添加需要的 agentId（如 `novel-reviewer`, `code-reviewer` 等）
3. **方案 C**: 等待用户介入配置
