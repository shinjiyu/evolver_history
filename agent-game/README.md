# Agent Game: 深渊遗迹探险队

一个 **AI Agent 驱动的协作探险游戏**。

## 🎮 游戏概念

多个 AI Agent 组成探险队，探索随机生成的地下遗迹。每个 Agent 有独特的角色、性格、能力和目标。他们需要协作应对挑战，同时追求个人使命。

**人类玩家的角色**：作为"观察者"和"顾问"，可以观看 Agent 的探险过程，提供建议，或在关键时刻接管某个 Agent 做出决策。


## 🎯 核心特性

### 1. 智能 Agent

每个 Agent 有：
- **性格**：勇敢、谨慎、贪婪、利他
- **目标**：个人和团队目标
- **记忆**：短期和长期记忆
- **关系**：与其他 Agent 的关系

### 2. 随机地牢

- 程序化生成地牢
- 多种房间类型：大厅、战斗、宝藏、陷阱、休息、神秘
- 动态敌人：根据等级调整难度
- 特殊事件：谜题、祝福、诅咒

### 3. 协作与冲突

**协作机制**：
- Agent 共享信息
- 支援队友
- 制定团队策略

**冲突机制**：
- Agent 有个人目标
- 性格差异导致分歧
- 关系好坏影响信任

### 4. 成长系统

- 经验值和升级
- 技能学习
- 装备收集
- 记忆积累

## 🤖 Agent 角色

### 战士 (Warrior) - 凯恩
- **性格**：勇敢（0.9），谨慎（0.3），贪婪（0.6），利他（0.5）
- **特点**：高 HP，高防御，保护队友
- **技能**：冲锋、盾击、嘲讽

### 法师 (Mage) - 艾莉丝
- **性格**：勇敢（0.5），谨慎（0.7），贪婪（0.4），利他（0.8）
- **特点**：高攻击，支援队友，魔法技能
- **技能**：火球术、治疗、护盾

### 盗贼 (Thief) - 罗格
- **性格**：勇敢（0.6），谨慎（0.5），贪婪（0.9），利他（0.2）
- **特点**：高速度，发现陷阱，收集宝藏
- **技能**：背刺、偷窃、侦查陷阱

## 📖 游戏流程

1. **初始化**
   - 生成地牢
   - 创建 Agent 探险队
   - 保存初始状态

2. **游戏循环**（每回合）
   - 感知阶段：更新 Agent 感知
   - 决策阶段：Agent 做决策（并行）
   - 执行阶段：执行动作（按速度排序）
   - 更新阶段：更新游戏状态
   - 记录阶段：记录事件

3. **游戏结束**
   - 所有 Agent 死亡
   - 击败 Boss
   - 达到最大回合数
   - 生成探险报告
